# Contributing

ThoughtVault is currently in the planning stage.

The best early contributions are:

- clarify requirements
- improve roadmap phases
- refine data model
- test document parsing approaches
- compare local vector databases
- design local AI prompts

## Project Principles

- local-first
- source-backed
- privacy-aware
- portable Markdown output
- AI suggestions require review before becoming durable knowledge

## Development Flow

1. Open an issue for the feature, task, or research question.
2. Keep changes small and traceable.
3. Update documentation when changing product or architecture decisions.
4. Do not add cloud dependencies as default behavior.
